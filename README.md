# Find Elements Automation Test

This is a simple **Selenium WebDriver** automation project using **TestNG**.

The project focuses on practicing how to:
- Find single elements using `findElement`
- Find multiple elements using `findElements`
- Interact with lists of web elements

## What This Test Does
- Logs in to https://www.saucedemo.com/
- Adds all products to the cart using `findElements`
- Removes all products from the cart

## Tools Used
- Java
- Selenium WebDriver
- TestNG
- EdgeDriver
